Project Details

Homepage

This is the homepage of the Student Management system. It serves as the central hub where users can navigate to other sections.

  • Allows access to different sections: Add student, Update student, View students
  • Provides an overview of the system's functionalities

Login Page

This page allows users to log into the system using their credentials.

  • Form for username and password
  • Authentication mechanism for secure login

Admin Dashboard

The admin dashboard allows administrators to manage the student records, view statistics, and more.

  • Shows a list of all students
  • Enables deleting or editing records
  • Displays basic statistics about the system
